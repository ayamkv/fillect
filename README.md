## how 2 do it urself 


• copy and paste this command <br>
> this will clone and install the required modules
```
git clone https://github.com/ayamkv/fillect.git
pip install -r requirements.txt
```

• fill in your api key in **config.py**


```
# your api keys
apikey = '#'
apikey_secret = '#'
token = '#'
token_secret = '#'
```

• then run 

```
python main.py
```
